🚀 Get RIFT Online — Free in 10 Minutes

Vercel backend + Netlify frontend. Both free forever. No credit card.

① CREATE THE BACKEND
1. github.com/new → create repo named rift-api
2. Add file api/chat.js — copy the code below
3. Add file package.json with content: {"name":"rift-api","version":"1.0.0"}
② DEPLOY ON VERCEL
1. vercel.com → Sign up free with GitHub
2. New Project → import rift-api
3. Environment Variables → add GEMINI_KEY = your key from aistudio.google.com
4. Deploy → you get rift-api.vercel.app 🟢
③ HOST THE RIFT FRONTEND
Easiest: netlify.com/drop → drag this HTML file → instant live URL
Alternative: Push to GitHub repo → enable GitHub Pages in Settings → free username.github.io/rift
All-in-one: Add index.html to your Vercel repo → it hosts both
④ CONNECT THEM
Open RIFT → Settings (⚙ in nav) → paste your URL:
https://rift-api.vercel.app/api/chat
Click Test Connection → if green, you're live globally 🌍
⚠ Never paste your Gemini API key directly in this HTML file. Always use Vercel's Environment Variables. It takes 30 seconds and keeps your key private.

api/chat.js — Copy This Exactly

// api/chat.js — Vercel serverless function with image vision support export default async function handler(req, res) { res.setHeader("Access-Control-Allow-Origin", "*"); res.setHeader("Access-Control-Allow-Methods", "POST, OPTIONS"); res.setHeader("Access-Control-Allow-Headers", "Content-Type"); if (req.method === "OPTIONS") return res.status(200).end(); const { prompt, image, mimeType } = req.body; const parts = []; if (image) parts.push({ inlineData: { mimeType: mimeType || "image/jpeg", data: image } }); parts.push({ text: prompt }); const r = await fetch( `https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-flash:generateContent?key=${process.env.GEMINI_KEY}`, { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify({ contents: [{ parts }] }) } ); const data = await r.json(); res.json({ text: data?.candidates?.[0]?.content?.parts?.[0]?.text || "No response" }); }
